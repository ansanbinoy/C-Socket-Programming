# C-Socket-Programming
## Installation
```
git clone https://github.com/ansanbinoy/C-Socket-Programming.git
cd C-Socket-Programming
gcc client.c -o client
gcc server.c -o server
```
## Usage
```
./server host port
./client host port
```
**Examples**
```
./server 127.0.0.1 8888
./client 127.0.0.1 8888
```
